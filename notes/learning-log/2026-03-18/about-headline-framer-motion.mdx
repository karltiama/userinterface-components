## Context

Landing page About section (`#about`) previously used a static h2. The goal was to cycle the ending phrase (‚Äúmatter.‚Äù ‚Üí ‚Äúexcite me.‚Äù ‚Üí ‚Äúhelp shape the future.‚Äù) with smooth motion.

## Goal

Implement rotating headline suffix using **Framer Motion** (`AnimatePresence` + `motion.span`) with sensible defaults for accessibility.

## Files Changed

- `package.json` / lockfile ‚Äî added **`framer-motion`**.
- `src/routes/landing.tsx` ‚Äî `AboutHeadline` component, wired into `AboutSection`.

## Implementation Notes (key logic explained)

- **Interval**: When the section is **in view** (`useInView`) and the user has **not** requested reduced motion, `setInterval` advances the suffix index every **3800ms**.
- **Exit / enter**: `AnimatePresence` with `mode="wait"` runs exit on the old span before the new one enters; avoids overlap.
- **`initial={false}`**: First visible phrase does not play an entrance animation on mount (only after first swap).
- **Reduced motion**: `useReducedMotion()` ‚Äî if true, cycling and Framer animations are skipped; static first phrase (**matter.**) is shown.
- **Stagger**: Existing `stagger(1, ‚Ä¶)` still applies to the `<h2>` (fade-in-up when section enters).

## Key Decisions (and alternatives considered)

| Decision | Alternative |
|----------|-------------|
| **framer-motion** | CSS-only keyframes + `key` remount ‚Äî fewer deps, less control over exit. |
| **Cycle only when `inView`** | Always cycle ‚Äî wastes updates when scrolled away. |
| **`aria-live="polite"`** on rotating span | Omit ‚Äî quieter for SR but less sync with visible text. |
| **Accent color on suffix** | All white ‚Äî less emphasis on the changing word. |

## Assumptions

- First suffix is acceptable as the only line when **prefers-reduced-motion** is on.
- Interval length (3.8s) is readable; tweak `ABOUT_HEADLINE_INTERVAL_MS` as needed.

## Failure Modes

- **Layout shift**: Longest phrase is ‚Äúhelp shape the future.‚Äù ‚Äî `min-h-[1.15em]` limits vertical jump; very narrow viewports may still wrap differently per phrase.
- **SR noise**: `aria-live="polite"` may announce each change; remove if it feels noisy for your audience.
- **`useReducedMotion()` null**: Framer may return `null` before hydration ‚Äî treated as falsy; cycling runs once resolved (if ever true, interval cleanup applies).
